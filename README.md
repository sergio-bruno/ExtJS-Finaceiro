# ExtJS-Finaceiro
Sistema com front-end em ExtJS e back-end em PHP

# Descrição do sistema
  Sistema voltado a controle de vendas e cadastro de e-mail e produtos, relatórios, contas a pagar e receber, emisão de boleto bancário.
  
# Requisitos
  Apache 
  PHP 5
  MySql 5
  ExtJS 4.2.1
  ZendFramework
  JasperReport
  FPDF
